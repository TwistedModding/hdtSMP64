FSMP's in-game configuration menu is built on the SKSE Menu Framework, a required
dependency you install separately (FSMP does not redistribute its DLL):
https://www.nexusmods.com/skyrimspecialedition/mods/120352
by A. Thiago Kaqiue. Its source is under the GNU LGPL v2.1, provided in the file
SKSEMenuFrameworkLicense.txt. The framework's SDK exposes Dear ImGui by Omar
Cornut, under the MIT license, provided in the file DearImGuiLicense.txt.

The monospace font shipped for the config menu's output panels (FSMPMono.ttf, in
SKSE/Plugins/Fonts) is Roboto Mono by Google, under the Apache License 2.0; its
license is beside the font, in FSMPMono.LICENSE.txt.
